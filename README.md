# Код диаграммы путешествия пользователя (User Journey Diagram)
```
journey
    title Путешествие пользователя в автономном приложении
    section Выбор темы
      Пользователь открывает приложение: 5: Пользователь
      Находит список доступных тем: 4: Приложение
      Выбирает интересующую тему: 5: Пользователь
    section Изучение контента
      Пользователь изучает теоретический материал: 5: Пользователь
      Открывает 3D-модель объекта: 4: Пользователь
      Взаимодействует с моделью (вращение, масштабирование): 5: Пользователь
    section Прохождение теста
      Пользователь запускает тест по изученной теме: 5: Пользователь
      Решает тестовые задания: 4: Пользователь
      Получает оценку и очки: 5: Приложение
    section Разблокировка нового контента
      Приложение начисляет очки прогресса: 5: Приложение
      Пользователь разблокирует новую тему: 4: Пользователь
    section Завершение
      Пользователь просматривает свои достижения: 4: Пользователь
      Закрывает приложение: 5: Пользователь
```

# Код квадрант-графа
```
graph TD
    A1(Квадрант 1 - Высокая важность, Низкая сложность)
    A2(Квадрант 2 - Высокая важность, Высокая сложность)
    A3(Квадрант 3 - Низкая важность, Низкая сложность)
    A4(Квадрант 4 - Низкая важность, Высокая сложность)

    A1 --> B1(Выбор темы)
    A1 --> B2(Просмотр теории)
    A1 --> B3(Начисление очков)

    A2 --> C1(Интерактивные 3D-модели)
    A2 --> C2(Разблокировка тем)

    A3 --> D1(Напоминания о новых темах)
    A3 --> D2(Отображение статистики)

    A4 --> E1(Сложные визуальные эффекты)
    A4 --> E2(Синхронизация с внешними ресурсами)
```

# Код гит-графа
```
gitGraph
    commit id: "Начальная версия приложения"
    branch development
    commit id: "Создание структуры приложения"
    commit id: "Реализация системы прогресса"

    branch feature/3D-models
    commit id: "Добавление интерактивных 3D-моделей"
    commit id: "Оптимизация визуализации моделей"
    checkout development

    merge feature/3D-models id: "Слияние 3D-моделей"

    branch feature/testing
    commit id: "Добавление тестирования для тем"
    commit id: "Оптимизация проверки ответов"
    checkout development

    merge feature/testing id: "Слияние функционала тестов"

    branch feature/progress
    commit id: "Реализация системы очков"
    commit id: "Добавление званий и достижений"
    checkout development

    merge feature/progress id: "Слияние функционала прогресса"

    commit id: "Подготовка к релизу"
```
